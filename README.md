# SmartGastro

Este proyecto es un sistema en consola hecho en Python para un foodtruck.
Permite cargar productos, registrar ventas y controlar stock.

## Funciones principales
- Agregar productos
- Registrar ventas
- Mostrar inventario
- Validar stock insuficiente

## Como ejecutar
1. Abrir terminal en la carpeta del proyecto
2. Ejecutar: python smartgastro.py
